struct discord_ret_guild¶
#include <discord.h>

Request’s return context.

Public Members

void (*done)(struct discord *client, void *data, const struct discord_guild *ret)¶

optional callback to be executed on a successful request

void (*fail)(struct discord *client, CCORDcode code, void *data)¶

optional callback to be executed on a failed request

void *data¶

user arbitrary data to be retrieved at done or fail callbacks

void (*cleanup)(void *data)¶

cleanup for when data is no longer needed

Note

this only has to be defined once, it shall be called once data is no longer referenced by any callback

bool high_p¶

if true then request will take priority over already enqueued requests

struct discord_guild *sync¶

if an address is provided, then request will block the thread and perform on-spot. On success the response object will be written to the address, unless enabled with DISCORD_SYNC_FLAG
